# Data directory

Данная директория используется для хранения **входных и вспомогательных файлов**,
которые применяются в задачах модуля **M2.7 — Контекстные менеджеры**.

## Содержимое

- `text.txt`
  Пример текстового файла для локальных экспериментов и тестирования работы
  контекстных менеджеров с файловыми ресурсами.

  Файл **не отслеживается системой контроля версий** и может свободно изменяться
  или пересоздаваться во время выполнения заданий.

- `.gitkeep`
  Используется для фиксации пустой директории `data/` в репозитории.

## Примечания

- Файлы в данной директории не являются частью публичного API модуля.
- Содержимое `data/` может отличаться у разных пользователей.
- Директория служит исключительно учебным и вспомогательным целям.
