# Task 03 — Error Boundaries

| Слой                  | Что делает                                             | Какие ошибки ловит                     | Что пробрасывает выше                 |
|-----------------------|--------------------------------------------------------|----------------------------------------|---------------------------------------|
| I/O слой              | обрабатывает ошибки связанные с файлом                 | FileNotFoundError, JSONDecodeError     | всё остальное                         |
| Parser/Validator слой | обрабатывает ошибки связанные с валидацией ключа limit | KeyError, ValueError, LimitNotPositive | доменные ошибки вверх                 |
| Boundary/UI слой      | выводит сообщения об ошибках для пользователя          | только доменные ошибки                 | не делает перевод, только отображение |
