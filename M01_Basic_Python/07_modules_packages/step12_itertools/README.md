### üìà –ë–µ—Å–∫–æ–Ω–µ—á–Ω—ã–µ –∏—Ç–µ—Ä–∞—Ç–æ—Ä—ã

| –§—É–Ω–∫—Ü–∏—è                | –û–ø–∏—Å–∞–Ω–∏–µ                                                         |
|------------------------|------------------------------------------------------------------|
| `count(start, step)`   | –ë–µ—Å–∫–æ–Ω–µ—á–Ω–∞—è –∞—Ä–∏—Ñ–º–µ—Ç–∏—á–µ—Å–∫–∞—è –ø—Ä–æ–≥—Ä–µ—Å—Å–∏—è (`start`, `start+step`, ‚Ä¶) |
| `cycle(iterable)`      | –ë–µ—Å–∫–æ–Ω–µ—á–Ω–æ –ø–æ–≤—Ç–æ—Ä—è–µ—Ç —ç–ª–µ–º–µ–Ω—Ç—ã `iterable`                         |
| `repeat(elem, n=None)` | –ü–æ–≤—Ç–æ—Ä—è–µ—Ç `elem` `n` —Ä–∞–∑ –∏–ª–∏ –±–µ—Å–∫–æ–Ω–µ—á–Ω–æ                          |

```python
from itertools import count, cycle, repeat

for x in count(10, 5):
    print(x)
    if x > 30:
        break

colors = cycle(['red', 'green', 'blue'])

print(next(colors))  # green

for x in repeat('A', 3):
    print(x)  # A A A
```

### üß© –ö–æ–º–±–∏–Ω–∞—Ç–æ—Ä–Ω—ã–µ –≥–µ–Ω–µ—Ä–∞—Ç–æ—Ä—ã

```python
from itertools import product, permutations, combinations

print(list(product([1, 2], ['A', 'B'])))
# [(1, 'A'), (1, 'B'), (2, 'A'), (2, 'B')]

# [('a', 'b'), ('a', 'c'), ('b', 'a'), ...]

print(list(combinations('abc', 2)))
# [('a', 'b'), ('a', 'c'), ('b', 'c')]
```

### üßÆ –§–∏–ª—å—Ç—Ä–∞—Ü–∏—è –∏ —Å—Ä–µ–∑—ã

| –§—É–Ω–∫—Ü–∏—è                               | –û–ø–∏—Å–∞–Ω–∏–µ                                   |
|---------------------------------------|--------------------------------------------|
| `islice(iterable, start, stop, step)` | –°—Ä–µ–∑ –∏—Ç–µ—Ä–∞—Ç–æ—Ä–∞ –∫–∞–∫ —É —Å–ø–∏—Å–∫–∞                |
| `compress(data, selectors)`           | –í—ã–±–∏—Ä–∞–µ—Ç —ç–ª–µ–º–µ–Ω—Ç—ã, –≥–¥–µ `selectors == True` |
| `filterfalse(func, iterable)`         | –û—Å—Ç–∞–≤–ª—è–µ—Ç —ç–ª–µ–º–µ–Ω—Ç—ã, –≥–¥–µ `func == False`    |
| `dropwhile(func, iterable)`           | –ü—Ä–æ–ø—É—Å–∫–∞–µ—Ç –ø–æ–∫–∞ `func == True`             |
| `takewhile(func, iterable)`           | –ë–µ—Ä—ë—Ç –ø–æ–∫–∞ `func == True`                  |

```python
from itertools import islice, compress, filterfalse

nums = range(10)
print(list(islice(nums, 2, 8, 2)))  # [2, 4, 6]

data = ['a', 'b', 'c', 'd']
selectors = [1, 0, 1, 0]
print(list(compress(data, selectors)))  # ['a', 'c']

print(list(filterfalse(lambda x: x % 2, range(6))))  # [0, 2, 4]
```

### ‚ö° –ö–æ–º–±–∏–Ω–∏—Ä–æ–≤–∞–Ω–∏–µ –Ω–µ—Å–∫–æ–ª—å–∫–∏—Ö –∏—Ç–µ—Ä–∞—Ç–æ—Ä–æ–≤

### üß† –†–∞—Å—à–∏—Ä–µ–Ω–Ω—ã–µ —Ñ—É–Ω–∫—Ü–∏–∏ ‚Äî —Å –ø—Ä–∏–º–µ—Ä–∞–º–∏

```python
from itertools import accumulate, starmap, pairwise, tee, groupby, batched
import operator

# accumulate: –∫—É–º—É–ª—è—Ç–∏–≤–Ω—ã–µ —Å—É–º–º—ã/–ø—Ä–æ–∏–∑–≤–µ–¥–µ–Ω–∏—è

# tee: –Ω–µ–∑–∞–≤–∏—Å–∏–º—ã–µ –∫–æ–ø–∏–∏ –∏—Ç–µ—Ä–∞—Ç–æ—Ä–∞

print(list(a), list(b))  # [0, 1, 2] [0, 1, 2]

# groupby: –≥—Ä—É–ø–ø–∏—Ä–æ–≤–∫–∞ –ø–æ–¥—Ä—è–¥ –∏–¥—É—â–∏—Ö —ç–ª–µ–º–µ–Ω—Ç–æ–≤ –ø–æ –∫–ª—é—á—É
animals = sorted(['cat', 'cow', 'dog', 'duck'])
for key, group in groupby(animals, key=lambda s: s[0]):
    print(key, list(group))  # c ['cat','cow']; d ['dog','duck']

# batched: —Ä–∞–∑–±–∏–µ–Ω–∏–µ –Ω–∞ —á–∞–Ω–∫–∏ —Ñ–∏–∫—Å–∏—Ä–æ–≤–∞–Ω–Ω–æ–≥–æ —Ä–∞–∑–º–µ—Ä–∞

```
